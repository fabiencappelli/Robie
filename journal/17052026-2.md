# Robie V1.2 — Premiers vrais composants fonctionnels

Aujourd’hui, l’objectif n’était pas de construire tout Robie V1.2, mais de commencer enfin à faire émerger un premier système cohérent, concret, vivant.

Le principe retenu a été simple :

```text
petites briques indépendantes
+
architecture propre
+
wins rapides et visibles
```

plutôt que :

```text
tout connecter d’un coup
```

Ce choix s’est révélé extrêmement utile.

---

# Ce qui a été réalisé aujourd’hui

## 1. Définition du socle architectural

Première décision importante :

```text
définir les classes et les contrats internes
avant de coder la logique
```

Le projet utilise désormais un fichier :

```text
classes.py
```

plutôt que `models.py`, afin d’éviter toute ambiguïté avec :

- modèles LLM
- modèles ML
- modèles d’embeddings

Le fichier centralise :

- `Book`
- `Library`
- `Intent`
- `StartMode`
- futures structures conversationnelles

---

# 2. Création du format de bibliothèque

Un premier format stable de `library.json` a été défini.

Exemple :

```json
{
  "version": 1,
  "books": [
    {
      "id": "rdf_04",
      "title": "L'île au secret",
      "series": "Les Royaumes de feu",
      "volume": 4,
      "path": "/home/fabien/audiobooks/rdf_04.mp3",
      "aliases": [],
      "language": "fr"
    }
  ]
}
```

Point important établi :

```text
versionner le format
≠
versionner le contenu
```

Le champ `"version"` sert à gérer les futures évolutions structurelles du fichier.

---

# 3. Validation Pydantic

La bibliothèque est désormais :

- chargée
- validée
- typée

via Pydantic.

Le système possède donc déjà :

```text
une vraie source de vérité structurée
```

pour les audiobooks.

---

# 4. Premier pipeline conversationnel fonctionnel

Premier pipeline réellement opérationnel :

```text
Intent structuré
    ↓
lookup bibliothèque
    ↓
génération phrase naturelle
```

Exemple :

```python
Intent(
    intent="play_audiobook",
    book_id="rdf_04",
    start_mode=StartMode.BEGINNING,
)
```

↓

```text
Tu veux écouter Les Royaumes de feu, tome 4, L'île au secret, depuis le début ?
```

---

# 5. Première synthèse vocale réelle

Ajout de :

```text
tts.py
```

avec :

- `pico2wave`
- `aplay`

Pipeline désormais fonctionnel :

```text
Intent
    ↓
phrase de confirmation
    ↓
TTS
    ↓
lecture audio réelle
```

Robie parle désormais réellement.

---

# Point architectural majeur confirmé

Le principe central retenu reste :

```text
Le LLM ne doit PAS être le moteur de recherche de la bibliothèque.
```

Architecture prévue :

```text
transcription
    ↓
matching local
    ↓
shortlist
    ↓
Qwen
```

et non :

```text
bibliothèque complète
→ LLM
```

---

# Pourquoi cette approche fonctionne bien psychologiquement

Un point important est apparu pendant le développement :

```text
les petits wins visibles changent complètement
l’énergie d’un projet long
```

Même si Robie ne comprend pas encore réellement la voix, il possède déjà :

- une bibliothèque réelle
- des structures stables
- un pipeline conversationnel
- une synthèse vocale
- une première personnalité technique cohérente

Le projet commence à “exister”.

# Point important retenu pour la normalisation

Normalisation légère uniquement :

```text
- lowercase
- suppression accents
- suppression ponctuation
```

mais PAS :

```text
quatre → 4
```

afin de conserver les erreurs phonétiques comparables aux sorties Vosk.
